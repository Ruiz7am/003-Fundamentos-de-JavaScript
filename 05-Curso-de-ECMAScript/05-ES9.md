# ECMAScript 9 (2018)

## Expresiones regulares

```javascript
const regex = /(\d{4})-(\d{2})-(\d{2})/;
const matchers = regex.exec('2025-01-01');
console.table(matchers);
```

(index) | values
:---: | :---: 
0 | '2025-01-01'
1 | '2025'
2 | '01'
3 | '01'
index | 0
input | '2025-01-01'
groups | undefined

## Spread

```javascript
const user = { username: 'ruiz7am', age: 35, country: 'MX' }
const {username, ...values} = user;
console.log(username);
console.log(values);
```

output:

```javascript
ruiz7am
{ age: 35, country: 'MX' }
```
